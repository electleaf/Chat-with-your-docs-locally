Developed an end-to-end RAG pipeline using Streamlit to provide a stateful, interactive interface for natural language querying of PDF datasets. 
Engineered an AI engine using LangChain and OpenAI API to automate data tasks including text chunking, embedding generation, and information retrieval.
Implemented Streamlit Session State for advanced state management, enabling persistent chat history and seamless data transition between processing and querying stages. 
Integrated ChromaDB for vector storage and semantic search, ensuring high data reliability and accuracy through rigorous data validation of retrieved context.
